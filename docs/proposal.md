# 行人意圖辨識與斑馬線 LED 預警提案研究報告

## 執行摘要
本案最適合定位為「115年人本環境全國大專院校學生競賽」B 組創意構想組的科技應用型提案。建議作品名稱採用：**先行一步：單鏡頭 AI 行人意圖辨識與斑馬線 LED 預警系統**。

## 技術架構 (MVP)
1. **攝影端 (ESP32-CAM)**: 透過 MJPEG/HTTP 串流影像，降低攝影端的運算負荷。
2. **邊緣運算端 (Raspberry Pi 5)**: 負責人形偵測、多目標追蹤、座標轉換、意圖分類及 GPIO LED 驅動。
3. **場域應用**: 定位於校園周邊無號誌或弱號誌行穿線，提升慢行安全。

## 核心設計原則
- **輕量化攝影端**: ESP32 只做影像採集。
- **穩定運算端**: Pi 5 集中處理 AI 邏輯。
- **直觀預警**: 提前點亮斑馬線 LED，彌補駕駛辨識落後。

## AI 判讀流程
1. 人形偵測 (YOLOv8n)
2. 多目標追蹤 (ByteTrack)
3. 地面座標轉換 (Homography)
4. 特徵萃取 (速度、向量、距路緣距離、停留時間)
5. 輕量分類器 (Waiting / Parallel / Intent / Crossing)
6. 時序平滑與 LED 驅動規則

## 開發時程
- **上傳版短衝 (4月30日前)**: 完成原型架構、構想書、A0 版面。
- **決審版深化 (7月18日前)**: 本地資料標註與分類器優化。
